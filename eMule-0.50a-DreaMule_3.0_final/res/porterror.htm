프루나가 사용하도록 지정된 통신 포트를 사용할 수 없습니다.
프루나가 사용하도록 설정된 통신 포트(TCP)를 이미 다른 프로그램이 사용 하고 있습니다.
일반적으로 이 문제는 다른 파일공유 프로그램이 실행되어 있거나 네트웍 문제로 인하여 발생할 수 있습니다.
문제 해결 방법
다른 파일 공유 프로그램과 동시 사용하는 경우
두가지 이상의 파일공유 프로그램을 동시에 사용하게 되면 통신 포트가 충돌을 일으켜 사용할 수 없는 경우가 있습니다.
이럴 경우에는 각각의 프로그램이 사용하는 통신 포트가 중복 되지 않도록 설정을 해야 합니다.
프루나 옵션 > 연결 > 고급 옵션에서 기본 포트를 변경하시기 바랍니다.
동시에 두가지 이상의 파일 공유 프로그램의 사용은 부득이한 경우를 제외하고는 권장하지 않습니다.
프루나만 사용시에도 발생을 하는 경우
시스템 문제로 인하여 해당 포트를 사용하지 못하는 경우가 있습니다. 이 경우에는 시간이 충분히 지난 후 다시 사용하거나 PC를 다시 껏다 켠 후에 사용해 보시기 바랍니다.
