프루나 사용시 주의사항
인터넷 또는 프루나에서 출처가 불분명한 파일을 받아 사용을 하는 경우 악성코드, 바이러스등 으로부터 피해를 보는 경우가 많이 있습니다.
사용자의 약간의 주의만으로도 악성코드, 바이러스등으로 인한 피해를 줄일수 있습니다.
문제 해결 방법
악성코드(스파이웨어)
인터넷을 사용할때 기본 웹페이지가 이상한 사이트로 연결이 되거나 웹페이지를 열면 이상한 창이 뜨는 경우를 많이 겪었을것입니다.
이 문제는 악성코드(스파이웨어)로 인한 것으로 직접적으로 프루나 사용이 문제가 되지는 않지만 인터넷이나 프루나를 통해 출처가 불분명한 파일을 받아 사용하는 경우 해당 파일에 있는 악성코드(스파이웨어)가 사용자가 모르게 설치되어 문제를 일으키게 되는 것입니다.
이러한 악성코드(스파이웨어)는 전용 프로그램으로 사용자가 정기적으로 PC를 검사하여 삭제를 하여야 합니다. 악성코드(스파이웨어) 전용 제거 프로그램 자료실 사이트등에서 받아서 설치를 하신 후에 사용하면 됩니다.
바이러스(웜바이러스)
프루나는 사용자간의 PC에 있는 파일를 직접 교환하는 프로그램입니다. 그렇기 때문에 바이러스가 걸린 파일이라도 그대로 교환이 될수가 있습니다.
바이러스에 걸리게 되면 바이러스를 치료한다고 해도, 시스템 파일등이 손상을 입어 아무런 이유없이 사용하는 프로그램이 이상 종료되거나 문제를 일으킬수 있습니다. 반드시 예방에 주의를 기울이는 것이 좋습니다.
바이러스 백신 소프트웨어가 설치되어 있으면 전송 받은 모든 파일에 대해 바이러스 검사를 실행할 수 있습니다.
파일 전송시 자동으로 바이러스 검사를 실행 하려면 [옵션], [받기] 탭을 차례로 클릭한 다음 바이러스 프로그램을 지정하세요.
그리고 바이러스 백신 소프트웨어는 실시간 검사 기능이 있다면 설정하도록 하세요.
pif파일이나 avi.exe, zip.exe등 확장자가 덧붙여지거나 변경된 파일등은 거의 대부분 바이러스입니다.
